# sample-mean

This is a study project to understand the concepts and development of a Angular 4 app using a MEAN (Mongo, Express, Angular, Node.js) stack.

The notable technologies used on this project are:

- Node.js
- Express
- Mongoose
- Socket.io
- ArgGIS
- Angular 4
- Angular Material
- MongoDB
